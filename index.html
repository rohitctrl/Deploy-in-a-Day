Hermes Agent

The self-improving AI agent by Nous Research. Gets smarter every session.

Built by Nous Research (the lab behind the Hermes model family), Hermes Agent is an autonomous task-execution framework that runs persistently on your server. Unlike session-based AI tools that start fresh every time, Hermes maintains three layers of memory: working memory for current tasks, long-term semantic memory for preferences, and episodic skill logs for recurring workflows. It writes reusable Skill Documents after solving complex tasks so it doesn't re-derive the same solution twice.

Runs comfortably on Oracle Free Tier's 24GB ARM instance. Supports Anthropic, OpenAI, and local models via Ollama. The kit includes model routing config that prevents cold-start memory issues on first boot.

Agents get roles, reporting lines, per-agent budgets, and task queues. An audit log traces every decision. A board approval workflow lets you approve agent hires and override strategy. It crossed 38K GitHub stars in under four weeks, one of the fastest-growing open-source AI projects of 2026.

The self-hosted only requirement makes Oracle VPS the ideal home for Paperclip. The kit configures the React dashboard to be accessible via a reverse proxy so you can monitor your agent org from any browser.
